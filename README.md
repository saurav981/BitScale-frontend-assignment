# Bitscale Grid

Data grid application built with Next.js 15, TypeScript, and Tailwind CSS.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (icons)

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Features

- **Sorting**: Click column headers or use Sort button
- **Filtering**: Search text + multi-select filters
- **Infinite Scroll**: Loads 10 rows at a time
- **Expandable Rows**: Click row/name/arrow to expand
- **Email Actions**: Click email status badges for actions
- **Responsive**: Table on desktop, cards on mobile
- **Sticky Elements**: Toolbar and table header stay visible
